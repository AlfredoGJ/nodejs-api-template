# Node.js REST API Template
Simple template for creating a Node.js REST API (with MongoDB)

## Endpoints
__Users:__
- GET /users
- GET /users/:id
- POST /users
- DELETE /users/:id
- PUT /users/:id

__Tasks:__
- GET /tasks
- GET /tasks/:id
- POST /tasks
- DELETE /tasks/:id
- PUT /tasks/:id

__Authenticate:__
- POST /authenticate
